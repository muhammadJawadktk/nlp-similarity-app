# Text Similarity Explorer

A Streamlit web app that uses the free pretrained model **all-MiniLM-L6-v2** to measure how similar sentences are to each other.

## Model Used
- **Name:** all-MiniLM-L6-v2
- **Source:** Sentence-Transformers (HuggingFace)
- **Cost:** Free and open source

## App Purpose
Enter any sentence as a query and compare it against multiple sentences. The app shows similarity scores, a bar chart, a heatmap, and a 2D PCA plot.

## Streamlit App Link
[Click here to open the app](https://your-app-link.streamlit.app)

## How to Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
